# Weex-Twitter

使用 Python 编写的爬虫，自动搜索 Weex 在 Twitter 上的相关内容并搜集数据。

## 目录结构

```plaintext
Weex-Twitter/
├── main.py               # 主程序入口
├── login.py              # 登录功能模块
├── search.py             # 搜索功能模块
├── requirements.txt      # 依赖库列表
├── README.md             # 项目说明文档
└── chromedriver.exe      # ChromeDriver 执行文件
